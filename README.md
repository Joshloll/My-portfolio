# Virtual Assistant Portfolio Website

A modern, responsive portfolio website designed for Virtual Assistants with skills in Canva design, Excel/data management, and IT support.

## Features

### 🎨 **Design & Layout**
- **Modern SaaS-inspired design** with clean, minimalist aesthetic
- **Responsive mobile-first layout** that works seamlessly on all devices
- **Smooth animations** including fade-ins, hover effects, and parallax scrolling
- **Professional color scheme** with gradient accents (blue/purple theme)
- **Clean typography** using Inter font family

### 📱 **Sections Included**
1. **Hero Section** - Eye-catching introduction with name, role, and CTAs
2. **About Me** - Professional summary with animated statistics
3. **Services** - Four service categories with card layout:
   - Virtual Assistance
   - Canva Design
   - Excel & Data Entry
   - IT Support
4. **Portfolio** - Grid layout with modal previews for project details
5. **Skills** - Visual representation with progress bars and tags
6. **Testimonials** - Client reviews with star ratings
7. **Contact** - Interactive contact form and multiple contact methods

### 🔗 **Contact Features**
- **Contact form** with validation and user feedback
- **Multiple contact methods**:
  - Email (mailto link)
  - WhatsApp (wa.me link)
  - LinkedIn profile
  - Facebook profile
  - Telegram
- **Floating contact button** with expandable options
- **Social media integration** with hover animations

### ⚡ **Interactive Elements**
- **Smooth scrolling navigation** with sticky header
- **Mobile hamburger menu** with animated transitions
- **Portfolio modal system** for detailed project views
- **Animated skill bars** that trigger on scroll
- **Hover effects** on cards, buttons, and links
- **Typing effect** for hero title
- **Parallax scrolling** effects

### 🛠 **Technical Features**
- **Performance optimized** with debounced scroll events
- **SEO friendly** with proper meta tags
- **Accessibility features** including keyboard navigation
- **Cross-browser compatible**
- **Print-friendly** styles
- **Clean, maintainable code structure**

## Customization

### 🎯 **Personalization Required**
Update the following placeholders with your information:

1. **Personal Information** (in `index.html`):
   - Replace "Your Name" with your actual name
   - Update contact email: `your.email@example.com`
   - Update phone number: `+1 234 567 890`
   - Update social media links:
     - LinkedIn: `https://linkedin.com/in/yourprofile`
     - Facebook: `https://facebook.com/yourprofile`
     - Telegram: `https://t.me/yourusername`

2. **Portfolio Content** (in `script.js`):
   - Update project descriptions and details
   - Add/remove portfolio items as needed
   - Customize project outcomes and metrics

3. **Testimonials** (in `index.html`):
   - Replace with actual client testimonials
   - Update client names and companies
   - Adjust ratings if needed

4. **Styling** (in `styles.css`):
   - Modify accent colors by updating gradient values
   - Adjust fonts and spacing as preferred
   - Customize animation speeds

## 🚀 **Getting Started**

1. **Local Development**:
   ```bash
   # Using Node.js serve (recommended)
   npx serve -s . -p 8000
   
   # Or using Python (if available)
   python -m http.server 8000
   
   # Or using PHP (if available)
   php -S localhost:8000
   ```

2. **Open Browser**:
   Navigate to `http://localhost:8000`

3. **Deployment**:
   - **Netlify**: Drag and drop the folder
   - **Vercel**: Connect your GitHub repository
   - **GitHub Pages**: Upload to a GitHub repository
   - **Any hosting**: Upload all files to your web server
